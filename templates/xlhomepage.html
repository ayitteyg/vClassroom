Named Ranges

We normally performs some operations on certain columns with data in the spreed sheet. Example: we have the following sales data and would like to sum the sales at cell C5 . We would usally go the cell C5 and enter =SUM(B5:B16). This is simple right, but what if we name the range we want to sum as sales_data and refer to it always.


HOW?

Select the data range you want to sum, ie: B5:B16. On your ribbon, go to Formulas> Named Manager > New. give a name to your data: like sales_data. click on the reffer to, to see if you selected the right data, then CLOSE. Now in cell C12, enter = sum(sales_data)


Editing

You can change the name later if you wish, by using the edit instead of new. you can increase the range for future entery, for instance, go the Named Manager, select the named range, go to refers to, instead of :B16 , increase it to :B16000. Now any sales you enter within that column range is captured


How it improves your daily task

Assuming you often have to refer to your sales data multiples times in different sheets in your workbook. Naming your data saves your from shuffling between sheets to locate your data and dragging it. Wherever, you are in your workbook, just enter =sum(sales_data)
if your data spans several columns and rows, you can equally name the whole range and refer to it anywhere in your workbook for further analysis

Data Validation / Drop down

While working on our spreed sheets, sometimes we would prefer getting some sort of drop down list of items so we can select from, or probably restrict a particular cell or range to accept only a specific type of data. This is where data validation comes in handy.

For instance you have a list of customer at range B4:B10. you want to assign a customer a certain locations, say Accra,Kumasi,Tema. you want to avoid copy and paste or typing every time. What do you do?


HOW?

select the cell or range you want to get the drop down list, in this case cell E5:E7, On the ribbon, go to
Data
Data Validation
Select List from Allow
click on source, move to customer list and select range B4:B10
If you had name your range Then at source, you could enter =(named_range) and that should work as well


Editing

To clear the data validation, following the procedure above, then click clear all from the dialog box, you can check the box to clear from all cells / range with that same settings.


How it improves your daily task

Aside giving you a handy drop down tool to select from, it guards you against certain unintended errors which may have gone unnoticed.
For instance try to assign Accra to daniel by typing daneil instead of daniel. It throws an error and would not allow it. you can try editting the error message at the data validation dialog box, Error Alert. to suit your own style


Simple Application

You have sales data for several years and would like to return sales value for a specific months within a specific year. So create a year list and month list. Afterwards, at cell J7, create a drop down list for year, at J8:J9, creates a dropdown list for months as well
Now you can select the year, startMonth, endMonth
